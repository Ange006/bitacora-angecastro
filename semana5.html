Introducción al
manejo de Github y Netlify

Aprendimos a usar Github y Netlify de forma rapida y efetiva.


Sesión 7 (09-04-2024)

GitHub es una plataforma en línea que facilita la colaboración en el desarrollo de software utilizando el sistema de control de versiones Git. Es ampliamente utilizada por desarrolladores de todo el mundo para alojar, revisar y gestionar proyectos de software de código abierto y privado. GitHub utiliza Git, un sistema de control de versiones distribuido, que permite realizar un seguimiento de los cambios en el código a lo largo del tiempo. Los desarrolladores pueden realizar commits para guardar los cambios, crear ramas para trabajar en nuevas características o correcciones, y fusionar ramas para combinar cambios.

Sesión 8 (09-04-2024)

Netlify es una plataforma de alojamiento y despliegue para aplicaciones web estáticas y dinámicas. Ofrece una variedad de servicios diseñados para facilitar el desarrollo, el despliegue y la administración de sitios y aplicaciones web modernas. Netlify es conocido por su soporte robusto para sitios web estáticos generados por herramientas como Gatsby, Hugo, Jekyll, entre otros. Sin embargo, también es compatible con aplicaciones web dinámicas y API, permitiendo la creación de sitios web con funcionalidades avanzadas. Netlify Functions te permite ejecutar código del lado del servidor sin necesidad de configurar ni administrar servidores. Esto es útil para agregar funcionalidades dinámicas a tu sitio web sin tener que implementar ni mantener una infraestructura de servidor tradicional.
